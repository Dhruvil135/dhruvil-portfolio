# Public Assets

## Files to Replace

### 1. Profile Photo
- **File name:** `profile-photo.jpg` (or `.png`)
- **Location:** Place your photo in this `public/` folder
- **Recommended size:** 400x400 pixels (square)
- **Format:** JPG or PNG
- **Note:** Make sure the filename matches what's in `src/data/portfolioData.ts` (currently set to `/profile-photo.jpg`)

### 2. Resume
- **File name:** `resume.pdf`
- **Location:** Already exists in this folder - replace it with your updated resume
- **Format:** PDF only

## How to Add Your Photo

1. Prepare a professional headshot (square format recommended)
2. Rename it to `profile-photo.jpg` (or update the path in `portfolioData.ts`)
3. Place it in this `public/` folder
4. The photo will automatically appear in the About section
